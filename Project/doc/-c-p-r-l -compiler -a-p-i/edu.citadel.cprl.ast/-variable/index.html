open fun emitBranch(condition: Boolean, label: String)

For Boolean expressions, this method emits the appropriate branch opcode based on the condition. For example, if the expression is a "<" relational expression and the condition is false, then the opcode "BGE" is emitted. The method defined in this class works correctly for most Boolean expressions, but it should be overridden for relational expressions.
